# Setup Mac Develoment Environment for Building KCC pages

## Prerequisites

### The system:
* Native Mac with x64 architecture
* macOS Sierra 10.12.6 (16G1114)

### iterm
* iterm2 - Use installer from: https://www.iterm2.com/
